## 项目初始化工作流

### 1. 安装项目依赖
```bash
npm install
```

### 2. 安装 Firebase 和 PWA 插件
```bash
npm install firebase vite-plugin-pwa -D
```

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 构建生产版本（可选）
```bash
npm run build
```

### 5. 预览生产版本（可选）
```bash
npm run preview
```
